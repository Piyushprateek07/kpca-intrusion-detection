# KPCA-Based Intrusion Detection System

## Overview
This project implements an Intrusion Detection System (IDS) using Kernel Principal Component Analysis (KPCA) combined with multiple machine learning classifiers.

## Features
- KPCA for dimensionality reduction
- Multiple classifiers (RF, XGBoost, KNN, etc.)
- Cost-based threshold tuning
- Decision Stability Index (DSI)
- Evaluation using MCC, Accuracy, F1-score

## Datasets Used
- CICIDS Dataset
- UNSW-NB15 Dataset

## Results
- High accuracy on CICIDS (~99.9%)
- Robust evaluation using MCC and DSI
- Stability analysis across clusters

## Technologies
- Python
- Scikit-learn
- XGBoost
- NumPy, Pandas
